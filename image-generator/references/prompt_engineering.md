# Prompt Engineering Reference

Detailed templates and examples for Nano Banana Pro image generation.

## Quick Prompt Checklist

**All images:** Subject + Composition + Lighting + Style + Mood
**Photography:** Add camera/lens + aperture + lighting setup
**Illustrations:** Add art style + medium + color palette
**Text:** Quote exact text + typography + placement
**Products:** Add materials + surface + props

| Futuristic | sleek, high-tech, neon, chrome, holographic, cyber |
| Natural | organic, earthy, raw, rustic, authentic, handcrafted |

## Lighting Vocabulary

| Type | Effect |
|------|--------|
| Golden hour | Warm, soft, long shadows, romantic |
| Blue hour | Cool, moody, cinematic twilight |
| Overcast | Soft, even, no harsh shadows |
| Rim/edge light | Subject separation, dramatic outline |
| High key | Bright, airy, minimal shadows |
| Low key | Dark, moody, dramatic contrast |
| Three-point | Professional, balanced, controlled |
| Natural window | Soft directional, authentic feel |

## Common Mistakes → Fixes

| Mistake | Fix |
|---------|-----|
| Keyword stacking: "cat, cute, fluffy" | Narrative: "A fluffy cat sitting on velvet armchair..." |
| Vague: "nice website" | Specific: "E-commerce product page, minimal design, forest green CTA..." |
| No context: "person at computer" | Context: "Developer working late, blue monitor glow, minimal desk..." |
| Conflicting: "minimalist with lots of details" | Commit to one direction |
| No technical specs | Add: "Shot with 85mm f/1.4, soft key light..." |
